Перед началом пользования программой, убедитесь, чтобы шрифт консоли был или "Consolas" или "Lucida console".
Изменить его можно зайдя в свойства консоли, для этого нужно кликнуть правой кнопкой мыши по окну консоли(просто кликните правой кнопкой мыши там, где написано название программы) и в появившемся меню выбрать пункт "свойства".
После этого откроется окно, в котором, выберите шрифт или "Consolas" (если такой имеется), или "Lucida console".
Нажмите "ОК", и, если не хотите повторять эту операцию каждый раз при новом запуске программы, в окне, которое появится после нажатия "ОК", выберите "сохранить свойства для других окон с тем же именем".
Все, программа готова к использованию. Все данные хранятся в файлах "enrollees.txt", "courses.txt", "relations.txt".
Чтобы загрузить существующие базы данных, всегда следите за тем, чтобы они были в одной папке с программой.И пусть удача всегда будет с вами! :3
